1. 運営者

AI秘書（運営者: 宇野想一郎）

2. 収集する情報

本サービスでは、以下の情報を収集します。

  • LINE User ID
  • Gmailアクセストークン（OAuth2による認証）
  • Googleカレンダー情報（OAuth2による認証）

3. 利用目的

収集した情報は、以下の目的でのみ使用します。

  • メールの要約・分類・返信案の生成
  • カレンダーの確認・予定の登録
  • LINEを通じたサービス提供

4. 第三者提供

収集した情報を第三者に提供することはありません。ただし、法令に基づく場合を除きます。

5. データの保管

アクセストークンは暗号化して保管し、サービス提供に必要な期間のみ保持します。ユーザーが退会した場合、関連するデータはすべて削除します。

6. ユーザーの権利

ユーザーはいつでも以下の権利を行使できます。

  • 保有データの開示請求
  • データの削除請求（退会）
  • LINEでのブロックによるサービス停止
